🔬 Anatomía de una Alerta

Conoce cada pieza antes de construir.

Una alerta de GrandLine UI se compone de cuatro piezas modulares que puedes combinar según lo que necesites comunicar. La única obligatoria es el contenedor base con la clase .alert.

1️⃣ CONTENEDOR

.alert

Base obligatoria. Define padding, sombra y animación hover.

2️⃣ ICONO

.alert-icono

Símbolo visual al inicio. Opcional pero recomendado.

3️⃣ CONTENIDO

.alert-contenido

Agrupa título (.alert-titulo) y texto (.alert-texto).

4️⃣ CERRAR

.alert-cerrar

Botón opcional para descartar la alerta.

📈 Niveles de Complejidad

De lo más simple a lo más completo, paso a paso.

Empieza con lo básico y ve agregando piezas según la importancia del mensaje. Cada nivel es útil para un caso diferente: un aviso rápido, una notificación con contexto o un mensaje destacado con acción de cierre.

NIVEL 1 · MINIMAL

Solo el contenedor y un texto. Ideal para avisos rápidos dentro de formularios o barras laterales.

Guardando cambios automáticamente.

CÓDIGO 
<div class="alert alert-in">
  <p class="alert-texto">Guardando cambios automáticamente.</p>
</div>
NIVEL 2 · CON ICONO

Se añade un icono visual que refuerza el tipo de mensaje. Perfecto para notificaciones estándar.

✓

Perfil actualizado correctamente.

CÓDIGO 
<div class="alert alert-ex">
  <span class="alert-icono">✓</span>
  <p class="alert-texto">Perfil actualizado correctamente.</p>
</div>
NIVEL 3 · COMPLETA

Todas las piezas activas: icono, título, texto y botón de cierre. Úsala para mensajes críticos que el usuario debe atender.

✕

Error crítico

No se pudo conectar al servidor.

CÓDIGO 
<div class="alert alert-pe">
  <span class="alert-icono">✕</span>
  <div class="alert-contenido">
    <h4 class="alert-titulo">Error crítico</h4>
    <p class="alert-texto">No se pudo conectar al servidor.</p>
  </div>
  <button class="alert-cerrar" type="button">✕</button>
</div>

🎨 Galería de Variantes

Las seis personalidades de tus mensajes.

Cambia solo la clase .alert-* para aplicar una paleta diferente. Cada variante está pensada para un tipo específico de comunicación según su color.

✕

🗺️ Cuándo usar cada variante

Guía rápida para elegir el color correcto según el mensaje.

Variante Color Úsala para... Ejemplo
.alert-ex Verde Confirmar acciones completadas "Perfil guardado correctamente"
.alert-pe Rojo Errores o fallos graves "No se pudo eliminar el archivo"
.alert-ad Naranja Avisar antes de una acción "Esta acción es irreversible"
.alert-in Azul Información general "Nueva actualización disponible"
.alert-pr Dorado Destacar mensajes importantes "Oferta por tiempo limitado"
.alert-se Dorado brillante Mensajes premium o especiales "Has desbloqueado un logro"
