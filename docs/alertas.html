Documentacion de Alertas

Mensajes visuales para comunicar información importante al usuario. Seis variantes de color, iconos, títulos y botón de cierre listos para usar.

Empezar ahora Ver referencia
⚓ Cómo usar una alerta

Toda alerta parte de la clase base .alert, que define la estructura común (padding, tipografía, sombras y animación hover). A partir de ahí, le sumas una variante de color para indicar el tipo de mensaje. 

<div class="alert alert-ex">
  <span class="alert-icono">✓</span>
  <div class="alert-contenido">
    <h4 class="alert-titulo">Exito</h4>
    <p class="alert-texto">Operacion completada.</p>
  </div>
</div>
i

Recomendación

Usa el ícono adecuado según el contexto: ✓ para éxito, ⚠ para advertencia, ✕ para error e i para información. Así el mensaje se entiende incluso sin leer el texto.

🎨 Variantes de Color

Seis estilos para comunicar diferentes tipos de mensaje.

Cada variante aplica un color de fondo tenue, un borde lateral resaltado y un color de texto con buen contraste. Las seis opciones son: éxito, peligro, advertencia, info, primaria y secundaria.

🧩 Estructura Interna

Los elementos que componen una alerta y cómo combinarlos.

Una alerta se compone de tres partes opcionales: un icono (.alert-icono), un contenido (.alert-contenido) que agrupa título y texto, y un botón de cierre (.alert-cerrar). Puedes usar solo lo que necesites.

VISTA PREVIA

Alerta simple solo con texto.

✓

Alerta con icono y texto.

⚠

Completa

Alerta con icono, título y texto descriptivo.

CÓDIGO 
<!-- Solo texto -->
<div class="alert alert-in">
  <p class="alert-texto">Alerta simple.</p>
</div>

<!-- Con icono -->
<div class="alert alert-ex">
  <span class="alert-icono">✓</span>
  <p class="alert-texto">Alerta con icono.</p>
</div>

<!-- Completa -->
<div class="alert alert-ad">
  <span class="alert-icono">⚠</span>
  <div class="alert-contenido">
    <h4 class="alert-titulo">Completa</h4>
    <p class="alert-texto">Descripción del mensaje.</p>
  </div>
  <button class="alert-cerrar" type="button">✕</button>
</div>

Clase Tipo Descripción
.alert Base Clase obligatoria para cualquier alerta
.alert-ex Variante Éxito (verde) — confirmaciones positivas
.alert-pe Variante Peligro (rojo) — errores y problemas

.alert-in Variante Información (azul) — datos generales

.alert-se Variante Secundaria (dorado brillante)
.alert-icono Elemento Icono decorativo al inicio de la alerta
.alert-contenido Elemento Contenedor que agrupa título y texto

.alert-texto Elemento Texto descriptivo del mensaje
.alert-cerrar Elemento Botón para ocultar la alerta
⚓ Código recomendado con Grid System

Combina las alertas con el sistema de rejillas (.container, .row, .col-*) para posicionarlas correctamente dentro de tu layout. 

<div class="container">
  <div class="row">
    <div class="col-12 col-md-8 col-lg-6">
      <div class="alert alert-ex">
        <span class="alert-icono">✓</span>
        <div class="alert-contenido">
          <h4 class="alert-titulo">Bienvenido</h4>
          <p class="alert-texto">Tu sesión inició correctamente.</p>
        </div>
        <button class="alert-cerrar" type="button">✕</button>
      </div>
    </div>
  </div>
</div>
