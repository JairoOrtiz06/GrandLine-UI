Botones

Elementos interactivos con estilo pirata legendario. Variantes sólidas, outline y tres tamaños listos para cualquier aventura en tu interfaz.

Empezar ahora Ver referencia
Cómo usar un botón

Todo botón parte de la clase base .btn, que define la estructura común (padding, tipografía, bordes). A partir de ahí, le sumas una variante de color y, si quieres, un tamaño. Las clases se combinan en el mismo elemento separadas por un espacio. 

<button class="btn btn-pr">Primario</button>
<button class="btn btn-o-ex btn-lg">Confirmar</button>
i

Recomendación

Usa el atributo nativo disabled para desactivar un botón. El framework aplica la opacidad y el cursor adecuados de forma automática.

Variantes Sólidas

Fondo completo con el color del tema. Ideales para acciones principales.

Las variantes sólidas son las más usadas en llamadas a la acción. Cada clase aplica color de fondo, texto con buen contraste y borde del mismo tono. Existen seis: primario, secundario, éxito, peligro, advertencia e info.

CÓDIGO

<button class="btn btn-se">Secundario</button>

<button class="btn btn-pe">Peligro</button>
<button class="btn btn-ad">Advertencia</button>
<button class="btn btn-in">Info</button>

Variantes Outline

Solo el borde del color, manteniendo el fondo transparente.

Se crean con el prefijo .btn-o- seguido de la abreviatura del color. Son perfectas para acciones secundarias que acompañan a una principal, como el clásico par "Aceptar / Cancelar".

<button class="btn btn-o-ad">Advertencia</button>
<button class="btn btn-o-in">Info</button>

Tamaños

Ajusta el botón al contexto.

Usa .btn-sm para tablas o toolbars, .btn-md como tamaño estándar y .btn-lg para CTAs principales.

VISTA PREVIA
CÓDIGO 
<button class="btn btn-pr btn-sm">Pequeño</button>
<button class="btn btn-pr btn-md">Mediano</button>
<button class="btn btn-pr btn-lg">Grande</button>

Deshabilitado

Atributo HTML nativo, sin clases extra.

Añade disabled al botón para bloquear la interacción. Funciona con variantes sólidas y outline por igual.

VISTA PREVIA
CÓDIGO 
<button class="btn btn-pr" disabled>Primario</button>
<button class="btn btn-ex" disabled>Éxito</button>
<button class="btn btn-o-pe" disabled>Peligro</button>

Combinando Clases

Mezcla variantes y tamaños según el contexto de tu interfaz.

Puedes acumular clases sobre el mismo elemento en cualquier orden, siempre que incluyas la base .btn. Además, funcionan sobre etiquetas <a> para que un enlace luzca como botón sin perder su función de navegación.

VISTA PREVIA
Enlace como botón Descargar
CÓDIGO 
<button class="btn btn-ex btn-sm">Guardar</button>
<button class="btn btn-o-pe btn-md">Cancelar</button>
<button class="btn btn-in btn-lg">Más información</button>
<a href="#" class="btn btn-pr">Enlace como botón</a>
<a href="#" class="btn btn-o-ex btn-lg">Descargar</a>

Código recomendado con Grid System

Combina los botones con el sistema de rejillas (.container, .row, .col-*) para que tu diseño se adapte a móvil, tablet y escritorio sin CSS extra. 

<div class="container">
  <div class="row">
    <div class="col-12 col-md-6 col-lg-4">
      <button class="btn btn-pr btn-lg">Zarpar</button>
      <button class="btn btn-o-pe btn-md">Cancelar</button>
      <button class="btn btn-ex btn-sm" disabled>Confirmado</button>
    </div>
  </div>
</div>
