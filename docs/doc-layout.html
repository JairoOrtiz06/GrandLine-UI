Layout

Sistema de contenedores y grilla responsive de GrandLine-UI.

Contenedores

Los contenedores ayudan a controlar el ancho del contenido y mantener una separacion lateral uniforme.

Clase Uso Descripcion
.container Contenido centrado Ancho de 90%, maximo de 1200px y padding lateral.
.container-fluid Contenido completo Ocupa el 100% del ancho disponible con padding lateral.

Sistema de grilla

La grilla usa filas flexibles con 12 columnas. Puedes combinar columnas base, tablet y escritorio.

col-12 col-md-6 col-lg-4
col-12 col-md-6 col-lg-4
col-12 col-md-12 col-lg-4
