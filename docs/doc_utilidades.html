Propósito

Las utilidades de GrandLine UI permiten aplicar estilos comunes sin escribir CSS adicional. Son ideales para ajustes rápidos de presentación, alineación, espaciado y composición.

Tipografía

Controla tamaño, peso y alineación del texto mediante clases reutilizables.

Ejemplo visual

Texto pequeño - .fs-1

Texto medio - .fs-3

Texto destacado - .fs-5 .fw-bold

Texto centrado con color oro

<p class="fs-5 fw-bold text-center text-oro">
  Título destacado
</p>

Espaciado

Usa márgenes y paddings para separar elementos sin crear reglas CSS nuevas.

Ejemplo visual
.mb-1
.mb-3
.pt-4 .pb-4
<section class="mt-5 mb-5">
  <div class="pt-3 pb-3">Contenido</div>
</section>

Flexbox

Organiza elementos en fila y controla su alineación horizontal y vertical.

Ejemplo visual
Inicio Centro Final 
<div class="d-flex jc-between ai-center gap-3">
  <span>Elemento 1</span>
  <span>Elemento 2</span>
</div>

Display

Clase Resultado
.d-none Oculta el elemento.
.d-block Muestra el elemento como bloque.
.d-inline Muestra el elemento en línea.
.d-md-none Oculta el elemento en pantallas medianas o menores.
.d-md-block Muestra el elemento como bloque en pantallas medianas o menores.
