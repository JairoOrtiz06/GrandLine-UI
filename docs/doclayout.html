Layout

El sistema de layout de GrandLine UI define la estructura base para organizar contenido mediante contenedores, filas y columnas responsivas.

Contenedores

Controlan el ancho del contenido principal y permiten centrar secciones.

.container .container-fluid

Filas

Agrupan columnas y activan el comportamiento flexible del sistema.

.row

Columnas

Distribuyen el contenido en un sistema de 12 columnas adaptables.

.col-* .col-md-* .col-lg-*

Contenedores

Los contenedores definen el ancho horizontal de una sección. Se usan como envoltorio principal antes de crear filas y columnas.

Clase Descripción Uso recomendado
.container Contenedor centrado con ancho del 90% y máximo de 1200px. Contenido principal, documentación, tarjetas y secciones comunes.
.container-fluid Contenedor de ancho completo. Secciones extendidas, fondos completos o bloques de pantalla completa. 
<main class="container">
  Contenido centrado
</main>

<section class="container-fluid">
  Contenido de ancho completo
</section>

Sistema de columnas

El grid trabaja con 12 columnas. Cada columna debe ir dentro de una fila para mantener la alineación y el espaciado correcto.

Ejemplo visual
.col-12
.col-6
.col-6
.col-4
.col-4
.col-4
<div class="row">
  <div class="col-6">Columna 1</div>
  <div class="col-6">Columna 2</div>
</div>

Layout responsive

Las columnas pueden combinar clases base, medianas y grandes para cambiar su distribución según el tamaño de pantalla.

Patrón recomendado
.col-12 .col-md-6 .col-lg-4
.col-12 .col-md-6 .col-lg-4
.col-12 .col-md-6 .col-lg-4 
<div class="row">
  <div class="col-12 col-md-6 col-lg-4">
    Contenido responsive
  </div>
</div>

Referencia rápida

Grupo Clases Función
Contenedor .container Centra el contenido con ancho máximo.
Contenedor fluido .container-fluid Ocupa todo el ancho disponible.
Fila .row Agrupa columnas con flexbox.
Columnas base .col-1 a .col-12 Define el ancho de columna en el sistema de 12 partes.
Columnas medianas .col-md-1 a .col-md-12 Ajusta columnas para pantallas medianas.
Columnas grandes .col-lg-1 a .col-lg-12 Ajusta columnas para pantallas grandes.
