Documentación de Footer

Pie de página con dos diseños temáticos: el clásico mapa del tesoro con columnas organizadas y el impactante cartel Wanted pirata.

Ver variantes Ver referencia
⚓ Cómo usar un footer

El footer se construye con una clase base .footer y una variante de diseño (.footer--mapa o .footer--wanted). Dentro colocas los elementos modulares: títulos, texto, listas de enlaces y una zona base. 

<footer class="footer footer--mapa">
  <div class="footer-columnas">...</div>
  <div class="footer-base">...</div>
</footer>
💡

Tip del Capitán

Las clases internas (.footer-titulo, .footer-lista, etc.) se adaptan automáticamente al diseño de la variante que elijas.

🗺️ Variante Mapa del Tesoro

Diseño clásico con columnas organizadas.

La variante .footer--mapa usa un grid de 4 columnas, un separador decorativo con ancla y fondos radiales que simulan iluminación. Ideal para sitios con muchas secciones.

🧩 Elementos Internos

Las piezas que puedes combinar dentro del footer.

El footer se compone de elementos modulares que se adaptan a la variante elegida. Úsalos solo si los necesitas.

📝 TEXTOS .footer-titulo .footer-texto

.footer-titulo para encabezados de columna y .footer-texto para párrafos descriptivos.

🔗 ENLACES .footer-lista .footer-link

.footer-lista es un <ul> que contiene enlaces con clase .footer-link.

📐 COLUMNAS .footer-columnas

Grid de 4 columnas para organizar secciones. Solo funciona con la variante .footer--mapa.

⚓ SEPARADOR .footer-separador

Línea decorativa con icono al centro. Solo para la variante .footer--mapa.

📍 ZONA BASE .footer-base

Parte inferior centrada para el copyright o información legal. Se adapta a cualquier variante.

📖 Referencia Completa

Todas las clases del componente footer.

Clase Tipo Descripción
.footer Base Contenedor principal obligatorio
.footer-titulo Elemento Título en mayúsculas con letter-spacing
.footer-texto Elemento Párrafo descriptivo
.footer-lista Elemento Lista de enlaces sin estilos por defecto
.footer-link Elemento Enlace con hover desplazante
.footer-base Elemento Zona inferior centrada con separador
.footer-columnas Elemento Grid de 4 columnas (solo con mapa)
.footer-separador Elemento Línea decorativa con icono (solo con mapa)
.footer--mapa Variante Diseño con columnas y fondos radiales
.footer--wanted Variante Cartel pirata con bordes dobles
.footer--oscuro Variante Fondo negro con borde dorado brillante
