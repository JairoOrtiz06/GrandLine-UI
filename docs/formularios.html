Documentacion de Formularios

Los formularios en GrandLine-UI están diseñados para ser claros, accesibles y altamente personalizables. Incluyen estilos para inputs, estados interactivos y variantes visuales.

Empezar ahora Ver referencia
Como estructurar un formulario

Para mantener consistencia y accesibilidad, agrupa cada campo dentro de .form-group, utiliza .label para las etiquetas y aplica la clase .input para los campos. 

<form>

  <div class="form-group">
    <label class="label">Nombre</label>
    <input type="text" class="input">
  </div>

  <div class="form-group">
    <label class="label">Correo</label>
    <input type="email" class="input">
  </div>

  <div class="form-group">
    <label class="label">Mensaje</label>
    <textarea class="textarea"></textarea>
  </div>

</form>
i

Recomendación

📝 Formularios del Navegante

Clases disponibles para inputs, selects, textarea y estados.

Clase Categoría Descripción
.input Base Estilo base para inputs de texto
.select Base Estilo base para selects
.textarea Base Estilo base para áreas de texto
.input-wanted Variante Estilo tipo cartel WANTED (tono antiguo)
.input-marine Variante Estilo marine (azul limpio)
.input-treasure Variante Estilo tesoro (dorado brillante)
.checkbox Control Checkbox personalizado
.radio Control Radio button personalizado
.form-check Layout Alinea checkbox o radio con su etiqueta
.input-ex Estado Input válido (borde verde)
.input-pe Estado Input inválido (borde rojo)
.form-msg-ex Mensaje Texto de éxito debajo del input
.form-msg-pe Mensaje Texto de error debajo del input
:focus Interacción Resalta el input al enfocarlo
:disabled Estado Desactiva el input con opacidad reducida
