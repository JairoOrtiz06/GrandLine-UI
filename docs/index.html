Sobre el Proyecto

GrandLine-UI es un framework CSS desarrollado desde cero con SASS, inspirado en el universo de One Piece. Incluye 7 componentes reutilizables, clases utilitarias generadas con mapas y loops, sistema de nomenclatura consistente con dos iniciales en español, y una identidad temática pirata con la fuente Cinzel para títulos y una paleta de colores del Grand Line.

Instalación Rápida

Sigue estos pasos para integrar GrandLine-UI en tu proyecto.

ⓘ

Paso 1: Incluye las fuentes de Google

En el head de tu HTML agrega el link de Cinzel e Inter.

ⓘ

Paso 2: Incluye el CSS del framework

Agrega el link a style.css después de las fuentes.

ⓘ

Paso 3: Comienza a usar los componentes

Ya puedes usar las clases del framework en tu HTML.
