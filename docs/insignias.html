Insignias

Pequeños distintivos para etiquetar, contar, destacar o marcar estados. Las insignias son la forma más rápida de añadir contexto visual a cualquier elemento.

Ver el cofre Ver referencia
🔨 Receta de una Insignia

Construir una insignia es como forjar un arma pirata: combinas piezas hasta conseguir el resultado deseado. Parte siempre de la clase base .badge y añade los ingredientes que necesites.

🧪 Ingredientes

Obligatorio
.badge

Estructura base de la insignia

Recomendado
.badge-ex

Variante de color (6 opciones disponibles)

Opcional
.badge-pi

Modificador de forma o tamaño

📖 Código Secreto del Pirata

Todas las clases disponibles del componente insignia.

Clase Categoría Descripción
.badge Base Clase obligatoria para cualquier insignia
.badge-pr Color Primaria (dorado)
.badge-se Color Secundaria (dorado brillante)
.badge-ex Color Éxito (verde)
.badge-pe Color Peligro (rojo)

.badge-in Color Información (azul)
.badge-pi Forma Convierte la insignia en píldora redondeada
.badge-lg Tamaño Tamaño grande para destacar
