Navbar

Barra de navegación responsive con menú hamburguesa sin JavaScript. Tres variantes de color, tres estructuras y soporte completo para dropdowns.

Ver estructura Ver referencia
⚓ Cómo usar una navbar

La navbar se construye con tres piezas: una marca (logo o nombre), un menú de enlaces y un botón hamburguesa que aparece solo en móvil. Todo funciona sin JavaScript gracias a un checkbox oculto. 

<nav class="navbar">
  <a href="#" class="nav-marca">...</a>
  <input type="checkbox" id="nav-main" class="nav-check">
  <label for="nav-main" class="nav-toggle">...</label>
  <ul class="nav-menu">...</ul>
</nav>
⚡

Magia CSS Pura

El menú hamburguesa funciona con .nav-check:checked ~ .nav-menu, un selector que muestra el menú cuando el checkbox está activo. Sin una sola línea de JavaScript.

📐 Variantes de Estructura

Cambia la disposición del contenido.

Además de los colores, puedes modificar cómo se organizan los elementos dentro de la navbar con estas variantes estructurales.

INVERTIDA .navbar--derecha
  • Inicio
  • Mapa
  • Contacto

VERTICAL .navbar--vertical
  • Inicio
  • Mapa
  • Contacto

CÓDIGO 
<!-- Logo a la derecha -->
<nav class="navbar navbar--derecha">...</nav>

<!-- Logo arriba, menu abajo -->
<nav class="navbar navbar--vertical">...</nav>

📂 Menú Dropdown

Submenús desplegables sin JavaScript.

Agrega la clase .nav-dropdown al item que debe tener submenú, y dentro coloca un .nav-dropdown-menu con los enlaces. El menú aparece al hacer hover con una animación suave y una flechita conectora.

📱 Comportamiento Responsive

Cómo se adapta la navbar a cada dispositivo.

🖥️ ESCRITORIO ≥ 768px
  • ✓ Menú horizontal siempre visible
  • ✓ Todos los enlaces alineados a la derecha
  • ✓ Dropdowns abren al hacer hover
  • ✓ Botón hamburguesa oculto
📱 MÓVIL < 768px
  • ✓ Menú oculto por defecto
  • ✓ Botón hamburguesa visible
  • ✓ Al tocar: menú vertical desplegado
  • ✓ Dropdowns como submenús indentados

📖 Referencia Completa

Todas las clases del componente navbar.

Clase Tipo Descripción
.navbar Base Contenedor principal obligatorio
.nav-marca Elemento Logo o nombre del sitio
.nav-check Elemento Checkbox oculto que controla el menú móvil
.nav-toggle Elemento Botón hamburguesa (solo móvil)
.nav-menu Elemento Lista principal de enlaces
.nav-item Elemento Cada ítem dentro del menú
.nav-enlace Elemento Enlace de navegación
.nav-enlace--activo Estado Resalta el enlace de la página actual
.nav-dropdown Elemento Ítem que contiene un submenú
.nav-dropdown-menu Elemento Lista desplegable del submenú
.navbar--oscuro Variante Fondo negro con borde dorado
.navbar--transparente Variante Fondo transparente para superponer
.navbar--derecha Estructura Invierte el orden (logo a la derecha)
.navbar--vertical Estructura Apila logo arriba y menú abajo
