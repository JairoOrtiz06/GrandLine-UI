Navbar

El timón de tu interfaz. La barra de navegación es el punto de partida de cualquier aventura digital, y la nuestra viene con tres rutas listas para zarpar y un menú hamburguesa que funciona sin una sola línea de JavaScript.

Ver variantes Ver referencia

🛰️ Vista Aérea

La anatomía completa de una navbar desde arriba.

Una navbar se construye con tres zonas principales: la marca a la izquierda, el menú de enlaces a la derecha y un botón hamburguesa que aparece solo en móvil. Todas las piezas trabajan juntas gracias a un checkbox oculto que controla la apertura del menú sin necesidad de JavaScript.

ZONA 1 .nav-marca

Logo o nombre del sitio

ZONA 2 .nav-toggle

Hamburguesa (móvil)

ZONA 3 .nav-menu

Lista de enlaces

ESTRUCTURA BASE 
<nav class="navbar">
  <a href="#" class="nav-marca">
    <img src="logo.png" alt="Logo">
  </a>

  <input type="checkbox" id="nav-main" class="nav-check">
  <label for="nav-main" class="nav-toggle">
    <span></span><span></span><span></span>
  </label>

  <ul class="nav-menu">
    <li class="nav-item"><a href="#" class="nav-enlace">Inicio</a></li>
    <li class="nav-item"><a href="#" class="nav-enlace">Docs</a></li>
  </ul>
</nav>

📱 Bitácora Responsive

Cómo se transforma la navbar al cambiar de barco.

El verdadero tesoro de esta navbar es que es 100% responsive sin JavaScript. En pantallas menores a 768px, el menú se colapsa en un botón hamburguesa controlado por un checkbox oculto. Cuando el usuario lo activa, el menú se despliega verticalmente.

🖥️ ESCRITORIO ≥ 768px

Menú horizontal completo, sin botón hamburguesa. Todos los enlaces visibles a la derecha de la marca.

  • ✓ Menú siempre visible
  • ✓ Layout horizontal
  • ✓ Toggle oculto
  • ✓ Enlaces separados por gap
📱 MÓVIL < 768px

Menú oculto por defecto, se activa al tocar el botón hamburguesa. Los enlaces se apilan verticalmente al 100%.

  • ✓ Hamburguesa visible
  • ✓ Menú colapsable
  • ✓ Layout vertical
  • ✓ Sin JavaScript
⚡

Magia CSS Pura

El selector .nav-check:checked ~ .nav-menu permite mostrar el menú cuando el checkbox está activo, sin necesidad de JavaScript. Es uno de los trucos más elegantes de CSS moderno.

📖 Diario de Bitácora

Todas las clases del componente navbar.

Clase Tipo Descripción
.navbar Base Contenedor principal obligatorio
.nav-marca Elemento Logo o nombre del sitio (soporta imagen)
.nav-check Elemento Checkbox oculto que controla el menú móvil
.nav-toggle Elemento Botón hamburguesa visible solo en móvil
.nav-menu Elemento Lista de enlaces principal
.nav-item Elemento Cada ítem dentro del menú
.nav-enlace Elemento Enlace de navegación
.nav-enlace--activo Estado Resalta el enlace de la página actual
.navbar--oscuro Variante Fondo negro con borde dorado brillante
.navbar--transparente Variante Fondo transparente para superponer sobre imágenes
