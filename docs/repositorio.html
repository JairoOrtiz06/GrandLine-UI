Cómo Clonar el Repositorio

Trae el proyecto a tu máquina en 3 pasos.

PASO 1 Clonar

Abre tu terminal y ejecuta el comando de git clone. 

git clone https://github.com/JairoOrtiz06/GrandLine-UI.git
PASO 2 Entrar

Navega al directorio del proyecto clonado. 

cd GrandLine-UI
PASO 3 Compilar

Compila el SCSS a CSS con el comando de sass. 

sass scss/main.scss css/style.css

Estructura del Proyecto

Cómo está organizado el código fuente.

El proyecto sigue una arquitectura modular donde cada componente tiene su propio archivo SCSS. Esto facilita el mantenimiento y permite agregar nuevas piezas sin romper las existentes.

ÁRBOL DE ARCHIVOS
GrandLine-UI/
│
├── css/
│   ├── style.css           ← CSS compilado listo para usar
│   └── style.css.map
│
├── scss/
│   ├── abstractos/
│   │   ├── _variables.scss ← Colores, tipografía, espaciado
│   │   └── _mixins.scss    ← Funciones reutilizables
│   │
│   ├── componentes/
│   │   ├── _botones.scss
│   │   ├── _alertas.scss
│   │   ├── _insignias.scss
│   │   ├── _tarjetas.scss
│   │   ├── _navbar.scss
│   │   ├── _tabla.scss
│   │   ├── _formularios.scss
│   │   └── _footer.scss
│   │
│   ├── layout/
│   │   ├── _container.scss
│   │   └── _grid.scss
│   │
│   ├── utilidades/
│   │   └── _utilities.scss
│   │
│   └── main.scss           ← Archivo principal
│
├── html/                   ← Páginas de documentación
├── img/                    ← Imágenes del proyecto
└── README.md

Cómo Contribuir

Únete a la tripulación de GrandLine UI.

Este proyecto es colaborativo y toda ayuda es bienvenida. Si quieres aportar nuevas funcionalidades, reportar errores o mejorar la documentación, sigue estos pasos.

FORK Y PULL REQUEST
  1. Haz un fork del repositorio
  2. Crea una rama con tu mejora: git checkout -b mejora
  3. Haz commit de tus cambios
  4. Haz push a tu fork
  5. Abre un Pull Request al repositorio original
REPORTAR BUGS

Si encontraste un error, abre un Issue en GitHub describiendo:

  • Qué estabas intentando hacer
  • Qué esperabas que pasara
  • Qué pasó realmente
  • Captura de pantalla si es posible
