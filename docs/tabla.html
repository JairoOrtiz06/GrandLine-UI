Documentacion de Tablas

Usa las clases de tablas de GrandLine-UI para mostrar datos con estilos piratas, variantes visuales y soporte responsive.

Como hacer una tabla responsive

Para que una tabla no rompa el diseno en pantallas pequenas, envuelvela con un contenedor que tenga overflow-x: auto. Ese contenedor permite scroll horizontal solo cuando hace falta. 

<div style="overflow-x: auto;">
  <table class="table table-striped table-hover">
    ...
  </table>
</div>
i

Recomendacion

Combina .container, .row y .col-12 para que la documentacion se adapte a movil, tablet y escritorio.
