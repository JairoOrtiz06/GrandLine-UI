Tarjetas

Contenedores versátiles para mostrar información agrupada. Perfectas para perfiles, productos, estadísticas o cualquier contenido que merezca destacar.

Ver plantillas Ver referencia

Disección Arqueológica

Las partes de una tarjeta, descubiertas capa por capa.

Una tarjeta completa tiene cinco zonas bien definidas. Cada una cumple un papel específico y puedes usar solo las que necesites. La única obligatoria es el contenedor .card.

CARD-HEADER

CARD-TITLE

CARD-TEXT — Aquí va el contenido descriptivo principal de la tarjeta.

CARD-FOOTER

.card-img

Imagen superior opcional

.card-header

Encabezado destacado en color primario

03
.card-body

Contenido principal. Alberga título y texto

04
.card-title + .card-text

Título dorado y párrafo descriptivo

05
.card-footer

Pie con acciones o información adicional

CÓDIGO COMPLETO 
<div class="card">
  <img src="..." alt="..." class="card-img">
  <div class="card-header">Encabezado</div>
  <div class="card-body">
    <h3 class="card-title">Título</h3>
    <p class="card-text">Descripción del contenido.</p>
  </div>
  <div class="card-footer">
    <span>Info</span>
    <button class="btn btn-pr btn-sm">Acción</button>
  </div>
</div>
