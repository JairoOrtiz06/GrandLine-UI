Utilidades

Clases auxiliares para aplicar estilos rápidos de texto, espaciado, colores, flexbox, display, bordes y sombras.

Uso de utilidades

Las utilidades permiten modificar elementos directamente desde el HTML sin crear reglas CSS nuevas. Son útiles para ajustes rápidos y consistentes.

5. Display

Clases para controlar la forma en que se muestra un elemento.

Clase Resultado
.d-none Oculta el elemento.
.d-block Muestra el elemento como bloque.
.d-inline Muestra el elemento en línea.
.d-md-none Oculta el elemento en pantallas medianas o menores.
.d-md-block Muestra el elemento como bloque en pantallas medianas o menores.
